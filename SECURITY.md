# Security Policy

## Supported Versions

This repository contains custom units (`.nts1mkiiunit` files) for the **KORG NTS-1 MKII** synthesizer.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in any of the custom units or this repository:

1. **Do not open a public issue** - Security vulnerabilities should be reported privately
2. **Open a GitHub Security Advisory**:
   - Go to the [Security tab](https://github.com/Abusername1989/KORG-NTS1-MKii-NTS1-MKII-OSC-/security/advisories/new) in this repository
   - Click "Report a vulnerability"
   - Provide details about the vulnerability

3. **Or contact via GitHub Issues** (if not critical):
   - Open a new issue with the `security` label
   - Describe the vulnerability clearly
   - Include steps to reproduce (if applicable)

## What to Report

Please report:
- Security vulnerabilities in the code or build process
- Malicious code or backdoors
- Issues that could compromise user devices
- Privacy concerns

## What NOT to Report

Please do NOT report:
- Feature requests (use regular Issues)
- Bug reports for non-security issues (use regular Issues)
- Questions about usage (use Discussions or Issues)

## Response Time

We will acknowledge your report within **7 days** and provide an update on the status within **30 days**.

## Disclaimer

These custom units are provided **"AS IS"** without warranty. Users are responsible for:
- Verifying the safety of units before use
- Backing up their NTS-1 MKII before installing custom units
- Using units at their own risk
